# Personal Portfolio Website | الموقع الشخصي

A modern, responsive portfolio website built with **HTML, CSS, and JavaScript** to showcase my skills, projects, and experience. The Projects section is dynamically loaded from GitHub repositories, making it easy to keep the portfolio up to date.

موقع شخصي حديث ومتجاوب تم تطويره باستخدام **HTML وCSS وJavaScript** لعرض المهارات والمشاريع والخبرات. يتم تحميل قسم المشاريع تلقائيًا من مستودعات GitHub، مما يجعل الموقع يتحدث تلقائيًا عند إضافة مشاريع جديدة.

---

## ✨ Features | المميزات

- Responsive Design — تصميم متجاوب
- Dark / Light Mode — الوضع الداكن والفاتح
- Arabic & English Support — دعم اللغتين العربية والإنجليزية
- Dynamic GitHub Projects — تحميل المشاريع تلقائيًا من GitHub
- Skills Section — قسم المهارات
- Contact Section — قسم التواصل
- Modern UI & Smooth Animations — واجهة حديثة وحركات سلسة

---

## 🛠️ Built With | التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript
- GitHub REST API
